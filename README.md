## 简介
基于[腾讯云SDK](https://github.com/TencentCloud/tencentcloud-sdk-python)的视频字幕识别OCR

## 依赖环境
1. 依赖环境：Python 2.7-3.7
2. 从腾讯云控制台开通OCR服务
3. 获取 SecretID、SecretKey

## 使用
### 相关配置
1. 配置Python2.7-3.7环境
2. 修改config.py中的`SECRET_ID`和`SECRET_KEY`
3. 下载视频
### 安装相关依赖
```
pip install -r requirements.txt
```
### 执行命令
```
python ocr.py
```
